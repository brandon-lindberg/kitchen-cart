## General info
Kitchen Cart connects local communities and food trucks in Tokyo through a marketplace mobile web app.
	
## Technologies
This project was created with:
* Ruby on Rails
* React.js
* HTML/SCSS
* Mapbox API
* Cloudinary API
* Google Maps API
* Heroku
